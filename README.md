# Onion or not?

A news headline guessing game.


### Get Started
```shell
git clone git@github.com:danstewart/onionornot.git
cd onionornot

# Serve on port 8000 with
./ctl.sh --serve

# Or build with
./ctl.sh --build
cp -r src/* /data/www/onion.danstewart.xyz/
```
