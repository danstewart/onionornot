# Onion or not?

A news headline guessing game.


### Get Started
```shell
# Install swc
npm i -g @swc/cli @swc/core

git clone git@github.com:danstewart/onionornot.git
cd onionornot

# Serve on port 8000 with
./ctl.sh --serve

# Or build with
./ctl.sh --build

# Build and deploy
./ctl.sh --deploy
```
